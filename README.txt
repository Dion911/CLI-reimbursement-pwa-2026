This is a mobile-friendly PWA wrapper around the HTML prototype.

IMPORTANT:
Service workers don't run on file://
Run a local server:

  python3 -m http.server 8080

Then open:
  http://localhost:8080/reimbursement_pwa_2026/

Install on iPhone (Safari):
- Open the URL
- Share -> Add to Home Screen

Data + receipts are stored in LocalStorage (this browser only).
